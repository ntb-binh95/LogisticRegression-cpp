# LogisticRegression-cpp
Logistic Regression from scratch with Breast cancer dataset.
## How to run
Logistic regression
```
make logicR
./logicR
```

Test error rate: 1.75%

Linear Regression
```
make main
./main
```

Score: 0.732945
