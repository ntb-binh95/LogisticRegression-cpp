# LogisticRegression-cpp
Machine Learning algorithm from scratch with various dataset.
## How to run
Logistic regression
```
make logistic
./logistic
```

Test Accuracy with Breast Cancer Dataset: 98.2%

Linear Regression
```
make linear
./linear
```

Score with House Price data: 0.732945

K-Nearest Neighbor
```
make main
./main
```

Test Accuracy with Iris Dataset: 96.7%
